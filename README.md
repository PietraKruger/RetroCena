O nosso projeto é um aplicativo de streaming com foco em produções de filmes brasileiros, permitindo e proporcionando um acesso mais fácil a conteúdos que já 
foram produzidos, mas que hoje estão em esquecimento e que também já não são mais acessados.
Esse projeto tem o intuito de valorizar a produção brasileira de filmes. Para sua construção, foram utilizados CSS, HTML e JavaScript.
